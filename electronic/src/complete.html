Pour modifier ce programme il est conseiller de modifier les fichiers dans le dossier 'src', puis de lancer 'sh ./hypertool.sh -c'.
Ce dernier va mettre à jouer le code HTML 'complete.html' regroupant tous les fichiers HTML dont le nom commence par une série de 3 chiffres (de 000_ à 999_ ).

Afin de regrouper toutes les ressovurces dans un fichier HTML 'src.html', utiliser l'option de commande '-i' 'sh ./hypertool.sh -i src'.

Le script hypertool à été créer à la va-vite pour restructurer ce projet. La première fonction était de séparer les blocks 'article' les uns des autres.
Pour cela l'option de commande est '-s' : 'sh ./hypertool.sh -s %fichier%.html'.
Pour modifier au passage la mise en fome des balises, il suffit d'ajouter un 'b' (pour beautify) : 'sh ./hypertool.sh -sb %fichier%.html'.
